## About Emoji Universe App
A Python‑based desktop application that puts all 5,225 Unicode emojis at your fingertips. Built with PyQt5 and the emoji library, it offers a clean interface with instant search and one‑click copying.

## Features
- 🌟 Full emoji library – All 5,225 standard emojis included

- 📂 Side panel navigation – Browse by category (smileys, nature, food, flags, etc.)

- 🔍 Search bar – Find any emoji by name or keyword instantly

- 📋 One‑click copy – Click any emoji → automatically copied to clipboard

- 🎨 User‑friendly UI – Clean, responsive, and intuitive design

**Limitations**
- ⚠️ Does not support custom or newly added emojis beyond the Unicode 15 standard
